# combine_pdfs
## Combine PDFs without uploading to a website

Download the scripts, put them in a folder C:/Tools/

The at the top of the .py script there are some windows hooks. Run them in an Admin powershell.

### To get the script to combine pdfs:
<ol>
 <li> Right click on 1 pdf.</li>
 <li> Then click "Show more options" </li>
 <li>Find and click Combine PDFs. (A file browser should show up)  </li>
 <li>Select all the pdfs you want to combine. </li>
 <li>Name the merged pdf as you please and click enter.  </li>
</ol>

**You should now have a Combined PDF :)**
